# Торговая система ZigZag PA Strategy

## Обзор

Это торговая система, реализующая стратегию Price Action с использованием ZigZag (PA Strategy V4.1). Система построена с использованием архитектуры микросервисов, включающей фронтенд на React/Vite и бэкенд компонент.

## Структура проекта

```
/workspace/
├── backend/                 # Бэкенд сервисы и API
│   ├── apps/               # Модули приложения
│   ├── conf/               # Конфигурационные файлы
│   ├── manage.py           # Скрипт управления Django
│   ├── requirements.txt    # Зависимости Python
│   └── Dockerfile          # Конфигурация контейнера бэкенда
├── frontend/               # Фронтенд приложение на React/Vite
│   ├── public/             # Статические ресурсы
│   ├── src/                # Исходный код
│   ├── package.json        # Зависимости Node.js
│   └── README.md           # Документация для фронтенда
├── docker-compose.yml      # Оркестрация нескольких контейнеров
└── ZigZag PA Strategy V4.1.docx  # Документация по стратегии
```

## Используемые технологии

- **Фронтенд**: React + Vite с современными возможностями разработки
- **Бэкенд**: Бэкенд на Python (вероятно Django, судя по manage.py)
- **Контейнеризация**: Docker и Docker Compose для деплоя
- **Стратегия**: ZigZag Price Action Strategy V4.1

## Инструкции по установке

### Предварительные требования

- Docker и Docker Compose
- Node.js (для локальной разработки фронтенда)
- Python (для локальной разработки бэкенда)

### Запуск приложения

1. Перейдите в директорию workspace
2. Соберите и запустите все сервисы:
   ```bash
   docker-compose up --build
   ```

### Локальная разработка

Для разработки фронтенда:

```bash
cd frontend
npm install
npm run dev
```

Для разработки бэкенда:

```bash
cd backend
pip install -r requirements.txt
daphne -v 2 -b 0.0.0.0 -p 8001 conf.asgi:application
```

## Особенности

- Современный фронтенд на React с Vite сборщиком
- Архитектура микросервисов для масштабируемости
- Контейнеризованный деплой для консистентности
- Реализация стратегии ZigZag Price Action
- Возможности торговли в реальном времени (в соответствии со стратегией)
